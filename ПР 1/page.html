История создания

Язык C был разработан в начале 1970-х годов Деннисом Ритчи в Bell Labs. Он был создан для системного программирования и, в частности, для операционной системы UNIX. C стал развитием более раннего языка B, который, в свою очередь, произошёл от BCPL. Изначально C предназначался для написания операционной системы UNIX на PDP-11.

Краткий обзор языка

C — это императивный, структурный язык программирования. Он относится к языкам среднего уровня, так как имеет как черты языков высокого уровня (типы данных, циклы), так и возможность работы на уровне железа (указатели, битовые операции).

Основные характеристики языка C:

  1. Статическая типизация
  2. Низкоуровневый доступ к памяти
  3. Процедурное программирование
  4. Переносимость кода

Язык C имеет относительно небольшой набор ключевых слов, что делает его достаточно компактным, но при этом очень мощным.

Синтаксис C основывается на блоках кода, которые определяются фигурными скобками {}. Программы состоят из функций, среди которых есть одна главная — main(). Важным элементом являются указатели, которые позволяют работать с адресами памяти.

Основные характеристики C
Характеристика Описание
Простая языковая база Многие существенные возможности вынесены в стандартную библиотеку, например математические функции или функции работы с файлами
Ориентация на процедурное программирование Это удобно с точки зрения использования структурного подхода к процессу программирования
Контроль над памятью программисты могут управлять памятью вручную, что позволяет оптимизировать использование оперативной памяти
Широкое использование C используется во многих проектах, включая операционные системы (например, Unix и Linux) и программное обеспечение для встроенных систем

Примеры кода

Пример 1: Вывод сообщения на экран


#include <stdio.h>

int main() {
    printf("Привет, мир!\n");
    return 0;
}

Пример 2: Простейшая функция


int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(5, 3);
    printf("Сумма: %d\n", result);
    return 0;
}

Пример 3: Работа с массивами


#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};
    int i;
    
    for (i = 0; i < 5; i++) {
        printf("%d ", numbers[i]);
    }
    printf("\n");
    
    return 0;
}
